# 产品定位

## 一句话定位

> 一个**安装即用、不要账号**的加密资产看板，专注小组件体验，本地优先，开源。

## 目标用户

- **加密货币持有者**：想随时看价格但不想每次都打开交易所 App
- **开发者 / 技术爱好者**：欣赏开源、隐私优先、克制设计的人
- **苹果股票 App 的羡慕者**：希望加密版有类似的"安静的精致"

## 不做什么

明确"不做"比"做什么"更重要：

- ❌ **不做交易撮合** —— 这是交易所的事，CryptoPulse 是看板
- ❌ **不做账户系统** —— 无注册、无登录、无云端
- ❌ **不做社交 / 评论 / 资讯流** —— 不是内容平台
- ❌ **不堆指标** —— K 线就是 K 线，不上 MA/MACD/BOLL 等十几条
- ❌ **不收订阅费** —— 完全免费，没有 paywall

## 五个差异点

### 1️⃣ 小组件 S 级体验

**痛点**：行情类 App 的小组件普遍粗糙——尺寸单一、刷新延迟、点击只能开 App 首页。

**我们怎么做**：
- 3 种尺寸（1×1 单币 / 2×2 多币 / 4×2 含趋势图）
- 30 秒前台 / 5 分钟后台的智能刷新
- 点击单个币种 → deep link 到对应详情
- 主题秒切（系统切深色模式 < 1 秒响应）
- 中东 RTL 语言适配

### 2️⃣ 本地优先（Local-First）

**痛点**：大部分 App 都要登录才能用，自选列表"上传到云"听起来贴心但其实让隐私敏感人群焦虑。

**我们怎么做**：
- 安装即用，零账号
- 自选币种存在 DataStore
- 设置导出 / 导入用 JSON 文件（用户自己存哪里）
- 隐私政策一句话："我们不收集任何数据"

### 3️⃣ 价格涟漪动效

**痛点**：数字看久了麻木，但行情软件那种"闪烁"又太吵。

**我们怎么做**：
- Compose Canvas 自绘
- 涨 → 极淡的绿色脉冲从数字向外扩散，200ms 渐变
- 跌 → 红色脉冲
- 横盘 → 完全无动效（安静）
- 用户可关

这是 App 名"CryptoPulse"的来源——价格是有脉搏的。

### 4️⃣ 极简 K 线

**痛点**：所有加密 App 的 K 线都像 1990 年代的股票软件，密密麻麻全是线。

**我们怎么做**：
- 默认只显示价格曲线（不是 K 线柱）
- 颜色：上涨绿、下跌红、二色断点
- 周期切换在底部，不在顶部（拇指可达）
- 拖动 → 半透明十字线 + 价格 tooltip
- 双指缩放（v2）

参考美学：Robinhood、苹果股票 App、Nike Run Club。

### 5️⃣ 工程师友好

**痛点**：技术圈对 SaaS App 普遍不信任，但开源 App 在加密领域少得可怜。

**我们怎么做**：
- 全开源，MIT License
- 数据源可换（默认 CoinGecko + Binance，用户可在设置切到自托管 API）
- API key 用户自己填，不走我们的代理
- 完整的技术博客系列伴随开发（每周一篇）

## 视觉风格

- **配色**：极简，主色 `#06B6D4`（cyan，象征液体/脉搏），辅色 `#10B981`（涨）/ `#EF4444`（跌）
- **字体**：系统默认（SF / Roboto），不引入第三方字体
- **图标**：Material Symbols Outlined（线性，不填充）
- **间距**：4 / 8 / 16 / 24 / 32 dp 网格
- **圆角**：8dp 标配，12dp 卡片，16dp 弹窗
- **阴影**：极少用，靠分隔线和留白区分层级

灵感板：
- Robinhood（极简金融）
- 苹果股票 App（克制）
- Things 3（小品类高完成度）
- Linear（开发者审美）
